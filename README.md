# Python Web Scraper

This is a simple Python web scraping project that extracts book titles from a website.

## Features
- Scrapes book titles from a website
- Uses Python requests and BeautifulSoup
- Beginner-friendly project

## Technologies Used
- Python
- requests
- BeautifulSoup

## Project Files
scraper.py – main scraping script  
requirements.txt – required Python libraries  
output.txt – example scraped output  

## How to Run

1. Install required libraries
pip install -r requirements.txt

2. Run the scraper
python scraper.py

3. The script will print book titles from the website.
